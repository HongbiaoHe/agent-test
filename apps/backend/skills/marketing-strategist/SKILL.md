---
name: marketing-strategist
description: Marketing strategy + planning router. Covers six task families — (1) brand-level positioning for 0→1 brands or repositioning (For-X-who-need-Y-we-are-Z-that-provides-W statement, 4Cs Company/Category/Competitors/Customers, Status Level 1-5); (2) Big Idea / concept pitching with 3 distinct creative routes (each rooted in a different C — Consumer / Category / Culture / Company); (3) text-only mood board (color palette with hex, texture vocabulary, typography direction, lighting mood, composition references — no AI image-gen credits burned upstream); (4) campaign-level planning (multi-asset, multi-phase, channel mix — Product Launch / Flash Sale / Traffic Engine Velocity / Premium Trust Funnel with KPI structure); (5) content narrative-model selection from 8 frameworks (PAS / AIDA / FAB / USP / 4P / BBE / Slide Effect / HSO) for carousels, sales pages, multi-shot ads; (6) cross-channel adaptation matrix (allocate one Big Idea across IG / XHS / TikTok / EDM / LP with per-channel tonal adjustments and copy rewrite direction — does NOT specify ratios / pixels, those belong to platform sub-skills). SKILL.md is a router — load zero or more sub-skills per task based on the brief; no sub-skill is forced.
---

## Sub-skill index

| File | When to load | Forced |
|------|--------------|--------|
| [big-idea-concept-pitching.md](./big-idea-concept-pitching.md) | Use when the user asks for concepts / ideas / directions ("give me 3 directions", "三个创意"), or when the brief is mid-to-low specificity and creative direction has not been pre-decided (brand campaign / Big Idea / launch). Produces 3 distinct creative concept routes (not variations of one idea), each with: concept name, central insight, 1-line creative platform, 3 tagline candidates, key visual direction, and tone notes. User picks 1 route; selected concept anchors downstream Production. Skip when the user already provides a complete storyboard / image prompt / locked concept, says "just generate, no need to pitch", or the task is purely production (e.g. "resize this image for IG Story"). | — |
| [brand-positioning.md](./brand-positioning.md) | Use when the brand is at 0→1 stage — no defined positioning yet — or when an existing brand needs repositioning for a new market / category. Produces a positioning statement (For X who need Y, we are Z that provides W; Unlike competitor C, we D), defines Status Level 1-5, and outputs the brand truth that anchors all downstream campaign work. Strategy-layer skill, runs UPSTREAM of Big Idea / Campaign Plan / Brand Soul Enforcer. | — |
| [mood-board.md](./mood-board.md) | Use after a Big Idea route is picked and BEFORE Image Producer / Video Producer fires. Produces a text-only visual direction board — color palette (with hex codes), texture / material vocabulary, typography direction, lighting mood, composition references, and keyword vocabulary list. Does NOT generate images (saves AI image-gen credits). Locks visual direction so all downstream production reads as one coherent campaign rather than disconnected outputs. | — |
| [campaign-plan.md](./campaign-plan.md) | Use when a brief calls for campaign-level planning (multi-asset, multi-phase, channel mix) — not a single ad. Body first classifies the funnel type (Traffic Engine / Premium Trust / Barbell Split), then routes to one of 4 scenario branches: Product Launch (Teaser → Launch → Sustain), Flash Sale / Mega Campaign (Pre-Sale → Peak → Last Chance), Traffic Engine Velocity (high-volume organic sprint), or Premium Trust Funnel (Education → Mechanism → Retargeting). Outputs funnel + channel + cadence + KPI structure. | — |
| [content-strategy-models.md](./content-strategy-models.md) | **Prefer to load before any creative deliverable** — image generation, video generation, copywriting, and strategy work. Selects from 8 marketing narrative models (PAS / AIDA / FAB / USP / 4P / BBE / Slide Effect / HSO) based on **product category × audience awareness level (model skeleton)**, then **applies the user-picked big-idea route's driver / insight / tone / visual direction as a coloring layer over the skeleton**, then produces two-layer phase directives that downstream producers (Image / Video / Copy) consume so each asset reads as a stage of one narrative simultaneously colored by the chosen creative direction. Works for: single image / single ad / single ≤15s video (use entry phase as the visual focus, or fit all phases inside the 15s clip), multi-shot videos that must split into multiple ≤15s clips (phases allocated across clips — see `video-script-framework`), carousels, sales pages, multi-touchpoint campaigns. Skip only for local edits, pure resize / format conversion, or raw Q&A with no creative deliverable. | — |
| [cross-channel-adaptation-matrix.md](./cross-channel-adaptation-matrix.md) | Use after Big Idea is picked AND BEFORE Production skills fire. Distributes one Big Idea into a channel-by-channel matrix — which channels, how many assets per channel, what tonal adjustments per channel, what copy rewrite direction per channel. Does NOT specify ratios / pixels / specs (those are owned by Platform Image Style / Platform Video Style — single source of truth). This skill is about ALLOCATION and ADJUSTMENT, not specifications. | — |
